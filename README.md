# 학생 건강검사 결과 확인 프로그램

학교 보건실에서 학생 건강검사 결과(BMI, 혈색소, 혈액검사)를 입력하면
교육부 공식 판정 기준과 비교해 즉시 결과를 보여주는 웹 도구입니다.

## 특징
- 이름, 학번 등 개인정보를 입력하거나 저장하지 않습니다.
- 입력한 수치는 브라우저를 닫으면 사라지며, 서버나 어디에도 전송/저장되지 않습니다.
- 정밀검사가 필요한 수치는 빨간색 + 화살표로 표시됩니다.

## 사용 방법
1. `index.html` 파일을 다운로드합니다.
2. 더블클릭해서 웹 브라우저로 엽니다. (인터넷 연결 불필요)
3. 항목별로 수치를 입력하면 결과가 자동으로 표시됩니다.

## 판정 기준 출처
교육부 「초·중·고등학교 학생 건강검사 분석」 보고서
(학교건강검사규칙 제5조제2항 관련 [별표 2] 및 같은 보고서 <표 Ⅰ-8> 혈액검사 이상 판정 기준)

### 1. BMI
- 이 도구는 성인 기준(18.5 / 23 / 25)을 사용한 단순 계산기입니다.
- ⚠️ 교육부 공식 판정은 소아·청소년 성장도표의 **연령별·성별 백분위수**를 기준으로 하며,
  이 도구의 성인 기준과 차이가 있을 수 있습니다. 정밀 판정용으로 쓰지 않도록 주의하세요.

### 2. 혈색소 (빈혈 검사)
- **검사 대상: 고등학교 1학년 여학생** (교육부 학생건강검사 규칙상 전체 학생 대상이 아님)
- 판정: 12.0 g/dL 미만 → 빈혈 의심 (성별 구분 없음, 단일 기준)

### 3. 혈액검사 (과체중·비만 학생 대상)
- **검사 대상: 초4·중1·고1 과체중 및 비만 학생** (전체 학생 대상이 아님)

| 항목 | 단위 | 정상 | 정상(경계) | 정밀검사 요함 |
|---|---|---|---|---|
| 공복혈당 | mg/dL | <100 | 100~125 | ≥126 |
| 총콜레스테롤 | mg/dL | <170 | 170~199 | ≥200 |
| HDL콜레스테롤 | mg/dL | >45 | 40~45 | <40 |
| 중성지방(TG) | mg/dL | <90 | 90~129 | ≥130 |
| LDL콜레스테롤 | mg/dL | <110 | 110~129 | ≥130 |
| AST(SGOT) | U/L | ≤45(10세 이상) / ≤55(10세 미만) | — | 초과 시 |
| ALT(SGPT) | U/L | ≤45 | — | >45 |

### 4. 소변검사 (전체 학년 대상)

| 항목 | 정상 | 정밀검사 요함 |
|---|---|---|
| 요단백 | 음성(–), 약양성(±) | 양성(1+~4+) |
| 요잠혈 | 음성(–), 약양성(±) | 양성(1+~4+) |

※ 요단백·요잠혈이 동시에 양성이면 신장 질환 동반 의심으로 더 강하게 표시됩니다.

※ 검진기관이 사용하는 시약 등의 차이로 자체 판정 참고치를 따로 갖는 경우에는
검사결과와 문진 등을 종합하여 자체 판정 참고치를 기준으로 판정합니다 (교육부 보고서 원문 안내).

## 앞으로 추가하고 싶은 항목 (계획)
- [ ] 시력 검사 (정상: 양안 나안시력 0.7 초과 / 요함: 한쪽이라도 0.7 이하)
- [ ] 혈압 (성별·연령별·신장대비 백분위수 기준 — 별도 표 필요)
- [x] 소변검사 (요단백, 요잠혈)

## 라이선스 / 주의사항
이 도구는 참고용 판정 보조 도구이며, 의학적 진단을 대체하지 않습니다.
실제 진료/상담 연계는 학교 보건 절차에 따라 별도로 진행해야 합니다.
